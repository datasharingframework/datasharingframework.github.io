BPMN

DSF-Team5/27/24Less than 1 minute

Introduction

The DSF uses BPMN 2.0 to model processes. Specifically, the Camunda 7 dialect from the Camunda Modeler. Modeling processes for the DSF requires this modeler or any other modeler which is able to produce the correct Camunda dialect.
