## DSF Hub
The Data Sharing Framework (DSF) Process Plugin Hub is a web application designed for searching and discovering DSF process plugins. This application is being developed as part of the DSF Community project. It provides users with a centralized platform to explore, evaluate, and utilize various process plugins within the DSF ecosystem, enhancing collaboration and efficiency across different domains.


::: info Marketplace for process plugins
To view already published Process Plugins, visit the [DSF Hub](https://hub.dsf.dev).
:::

## How to publish a project on DSF hub

To add a new process plugins to the DSF hub you are required to email dsf-gecko@hs-heilbronn.de with the following information:
- name of the plugin
- a contact person
- the URL to the GitHub repository

We look forward to your contributions!
