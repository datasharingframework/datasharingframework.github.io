Resource Profile: DSF OrganizationAffiliation

Official URL: http://dsf.dev/fhir/StructureDefinition/organization-affiliation Version: 2.0.0
Active as of 2025-05-26 Computable Name: DSF OrganizationAffiliation

An OrganizationAffiliation to describe a parent / participating organization relationship in the DSF context with restrictions to enforce DSF specific details.

Usages:

  • CapabilityStatements using this Profile: CapabilityStatement[http://dsf.dev/fhir/CapabilityStatement/dsf]
  • This Profile is not used by any profiles in this Implementation Guide
