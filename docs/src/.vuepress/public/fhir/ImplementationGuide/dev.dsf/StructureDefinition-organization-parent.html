Resource Profile: DSF Parent Organization

Official URL: http://dsf.dev/fhir/StructureDefinition/organization-parent Version: 2.0.0
Active as of 2025-05-26 Computable Name: DSF Parent Organization

A Parent Organization as Meta Organization (e.g., Consortia) in the DSF context with restrictions to enforce DSF specific details.

Usages:
